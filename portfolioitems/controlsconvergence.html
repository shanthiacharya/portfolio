GE PlantWorks

Controls Application UX


Background

GE Controls sells hardware and software tools for Plant Automation. Current software controls tools do not allow for systems to integrate quickly and easily. Customers need to handle multiple tools, versions and their issues upgrades. Many engineering hours are wasted in setting up systems and requisition effort.



Goals

Create an integrated tool suite that provides a collaborative workspace and reduces effort and rework in controls SW engineering, controls SW integration and testing, and commissioning.



My Role

Research Participation, Interaction Design Lead and Prototyping, Collaboration with visual designer.



APPROACH

Showcase the future product vision to get customer validation and feedback. We moved from sketch to code quickly and created a site that provided a clear vision and roadmap.



Final Design

Final design created in code based on the GE Marketing Design System.



Outcome

This saved as a great tool to collect feedback and build consensus with the vision.
